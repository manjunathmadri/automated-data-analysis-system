# Automated Data Analysis System

A Python + MySQL based system that:

- Imports CSV datasets
- Stores data in database
- Generates sales reports
- Exports PDF reports
- Displays dashboard UI

## Tech Stack
- Python
- MySQL
- Pandas
- Flask
- ReportLab

## Features
✔ CSV Data Import  
✔ Database Integration  
✔ Automated Reports  
✔ PDF Export  
✔ Web Dashboard
